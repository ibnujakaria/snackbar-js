# snackbar-js
A simple material-design-styled snackbar you can use in any page of your projects.

<img src="https://media.giphy.com/media/xUPGGzh2MOoRPuwSe4/giphy.gif"/>

## Installation

npm
```
npm install snackbar-js
```

## How to Use
```
Snackbar
  .create()
  .setText('Hello world! :)')
  .setStyle('center')
  .show()
```
